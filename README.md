# 経営統合システム UIカンプ（keiei-togo v4.3）

株式会社Initial State / ビジプラの経営統合システム（14モジュール）の **HTMLカンプ** です。
静的HTML＋CSS＋少量のJSのみで動作します（ビルド・サーバー不要）。

## 見かた

1. `login.html` を開く（公開URLのトップ）
2. **ログインするメンバーを選択**（ロールにより表示メニュー・可視範囲が変わります）
   - 中田（管理者）／北山（経営・営業）／今井（経理）／幸田（コンサル部長PM）／新谷（営業・一般）／新垣（アシスタント）
3. サイドバー下部のユーザーカードから**アカウント切替**・ログアウト
4. **⌘K / Ctrl+K** で横断検索（会社・案件番号 IS-2026-…・画面・ルール）

## 注意

- **数値はすべてサンプル**です（実データではありません）
- モーダルの確定ボタンは動作しますがデータは変わりません（トーストで通知）
- 判定・しきい値の共通認識は `rules.html`、デザイン基準は `styleguide.html`

## 資産の更新ルール

`assets/style.css` `assets/ui.css` `assets/app.js` を変更したら、
全HTMLの参照クエリ `?v=43` の番号を上げてください（キャッシュ事故防止）。

- システム要件：要件定義書 v1.0（14モジュール）＋引き継ぎ書の反映待ち差分
